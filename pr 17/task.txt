Задание

Написать скрипт, для практической 16, который будет запоминать авторизованного пользователя, с помощью механизма сессий. Добавить кнопку выйти из аккаунта, которая уничтожает сессию.

В отчёте предоставить код PHP.

Подключение к БД организовать по следующим параметрам: localhost, root, '', pract*номер варианта*

При выкладывании работы в облако, приложить экспортированную sql базу.
